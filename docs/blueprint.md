# **App Name**: StudyGeniusAI

## Core Features:

- AI-Powered Summarization: Generate concise summaries of uploaded documents or pasted text using AI, highlighting key concepts and main ideas.
- AI-Driven Quiz Creation: Automatically generate quizzes from summaries or uploaded documents, with multiple-choice, true/false, and fill-in-the-blank questions.
- AI Flashcard Generation: Create flashcards with key terms and definitions from provided texts, optimized for efficient learning and memorization.
- Semantic Search and Import: Enable users to search for previously created summaries, quizzes, and flashcards based on semantic similarity using vector embeddings, facilitating content reuse.
- User Item Management: Store all generated content (summaries, quizzes, flashcards) with associated metadata in a Supabase database for persistent storage and retrieval.
- Recents Display: Display recently accessed items in a sidebar for quick access.
- Browser Storage: Temporarily save the user data (like recent quizzes) into the browser if the user is not logged in, and save it into the supabase later on login

## Style Guidelines:

- Primary color: Deep purple (#673AB7), evoking a sense of intellect, focus, and sophistication suitable for a study application.
- Background color: Light gray (#F5F5F5), providing a clean and distraction-free environment for reading and interacting with content.
- Accent color: Bright teal (#00BCD4), used for interactive elements like buttons and links, offering a vibrant contrast and signaling interactivity.
- Body font: 'Inter' (sans-serif) for clear, modern readability.
- Headline font: 'Space Grotesk' (sans-serif) to add a futuristic and intellectual feel, clearly setting apart headlines.
- Code font: 'Source Code Pro' (monospace) for displaying code snippets.
- Use outline icons related to studying, learning, and knowledge acquisition. Keep the style consistent and minimalist.
- Maintain a clean and organized layout with clear sections for content, controls, and navigation.
- Incorporate subtle animations, such as fading transitions when loading new content or generating summaries, to enhance user engagement.
